FAQ

Can we brand this page?

Yes — add your colors/logo, and we can also match the survey styling (where supported) to align with branding.

Is GitHub Pages good enough for a demo?

For a demo: yes. It’s static, simple, and reliable. For production customer use, you’d typically host on the customer’s site.

What if the embed doesn’t appear?

Usually it’s a mismatch in the allowed domain, or the script snippet isn’t included on the served page. GitHub Pages uses HTTPS so it’s rarely the problem.
